# Volaticus19
storing code

you need python(version 3.7 or below), pip, and tesseract(4.x)
they need to be added to the path in environment variable.
if they are in the path, you can type 'python' or 'pip' or 'tesseract' in terminal and get their details.

next, run 'pip install -r requirements.txt' in this directory. this help you get the modules needed to run the script.
if you get all modules successfully, you can try the script by 'python test.py'
